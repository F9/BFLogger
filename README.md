## Installation

BFLogger is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'BFLogger'
```

Add your url to post log online using this keyword on the info.plist

```
BFLogger_base_url
```
